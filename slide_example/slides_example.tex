\begin{frame}{Steiner Tree Problem}
Steiner tree problem是一個組合最佳化問題，與最小生成樹相似，是最短網路的一種。最小生成樹（minimum spanning tree, MST）是在給定的點集和邊中尋求最短網路使所有點聯通。而最小 Steiner tree 允許在給定的點外增加額外的點，使生成的最短網路開銷最小。\end{frame}
