#Newton's method#
Implement of Newton's method with MATLAB

##Function##
###Define###

````Matlab
mynewton('function of x',initial position,times)
````

###Example###

````Matlab
>> mynewton('tan(x)-x',3.14*3/2,100)

ans =

    4.493409457909064

````
